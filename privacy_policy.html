Политика конфиденциальности

Последнее обновление: 2026-05-22

Свалко — неофициальный мобильный клиент для сайта svalko.org. Настоящая политика описывает, какие данные обрабатывает приложение.

Данные, хранящиеся на устройстве

Приложение сохраняет на устройстве пользователя:

  • Настройки интерфейса (тема, язык).
  • История голосований за посты — исключительно локально, для предотвращения повторного голосования.
  • HTTP-кэш страниц сайта для ускорения загрузки.

Эти данные не передаются третьим сторонам и хранятся только на устройстве.

Взаимодействие с svalko.org

При использовании приложения запросы отправляются на сервер svalko.org для загрузки контента, а также при голосовании за посты. Обработка этих данных регулируется политикой конфиденциальности svalko.org.

Отчёты об ошибках

При возникновении необработанных ошибок приложение может отправить анонимный отчёт, содержащий описание ошибки, версию приложения и общую информацию об устройстве (модель, версия Android). Отчёт не содержит личных данных пользователя.

Данные, которые мы не собираем

Приложение не собирает и не передаёт: имена, адреса электронной почты, геолокацию, идентификаторы для рекламы или любые другие персональные данные.
